# Databricks Examples - Preparation guides and material for certification

In this repository I have built a range of examples that are directly based on the exam guidelines for the different Databricks certifications. The purpose of this repository is _not_ to provide you with a set of practice questions, but to demonstrate to you the intuition needed in order to answer the questions in the Databricks certification exam.
